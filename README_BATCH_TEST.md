# 设备投放支援模型 - 批量测试工具

## 🚀 快速开始

### 1. 生成测试清单

```bash
# 使用默认地址 (http://127.0.0.1:5000)
python3 batch_test.py

# 或指定服务器地址
python3 batch_test.py http://192.168.1.100:5000
```

### 2. 执行浏览器自动化测试

在 Cursor 中告诉 AI 助手：
```
使用浏览器工具（1440x900窗口），按照 batch_test.py 生成的清单，
依次访问每个 URL，等待'推理中...'消失后全页截图保存。
```

## 📁 文件说明

- `batch_test.py` - 批量测试脚本（主脚本）
- `screenshots/equipment_delivery/` - 截图保存目录
  - `test_manifest.json` - 测试清单（JSON格式）
  - `TEST_REPORT.md` - 测试报告
  - `01_*.png` - 场景截图

## 🎯 测试场景

共 6 个设备投放支援模型测试场景：

1. **任务编组** - 设备类型、数量和投放方式的编组策略
2. **高精度目标定位** - 视觉识别和精确投放点定位
3. **自主装卸控制** - 无人平台自主装载与卸载控制
4. **投放确认** - 投放结果的自动确认机制
5. **投放点环境安全性评估** - 投放点的环境风险评估
6. **投放流程异常检测与应急策略生成** - 异常检测和应急策略
